This Web service establishes a REST API for the creation and storage of a funding organization.

Web Service Usage:

  1. GET / - This message
  2. POST /
    • * name = string
    • logo = image
    • emailAddress = string
    • description = string
    • url = string
    • phoneNumber = string
    • deliveryPoint = string
    • city = string
    • administrativeArea = string
    • postalCode = string
    • country = string
    * = mandatory
  3. PUT / (Accepts updates of these fields via PUT)
    • name = string
    • logo = image
    • emailAddress = string
    • description = string
    • url = string
    • phoneNumber = string
    • deliveryPoint = string
    • city = string
    • administrativeArea = string
    • postalCode = string
    • country = string

    Result - Funding organization will be created or updated. JSON response will indicate success or failure with error.
    Authentication is required to use this feature.
